Gerar Aplicativo APK

Instruções para criar o aplicativo móvel

📱 Opção 1: Apache Cordova

1. Instalar Node.js e Cordova:

npm install -g cordova

2. Criar projeto:

cordova create motaxi com.motaxi.app "Mó Táxi"

3. Adicionar plataforma Android:

cordova platform add android

4. Copiar código HTML para www/

Substitua o conteúdo da pasta www/ pelo código do aplicativo

5. Gerar APK:

cordova build android

🚀 Opção 2:Capacitor (Recomendado)

1. Instalar o capacitor:

npm install @capacitor/core @capacitor/cli

2. Inicializar o projeto:

npx cap init "Mó Táxi" "com.motaxi.app"

3. Adicionar Android:

npx cap add android

4. Copiar arquivos:

npx cap copy

5. Abrir no Android Studio:

npx cap open android

⚠️ Requisitos Importantes:

  • • Android Studio: Necessário para compilar o APK
  • • Java JDK 8+: Ambiente de desenvolvimento Java
  • • Android SDK: Kit de desenvolvimento Android
  • • Gradle: Sistema de build (incluído no Android Studio)

📍 Configurações para Angola:

  • • Localização: GPS configurado para coordenadas de Angola
  • • Moeda: Kwanza (Kz) como padrão
  • • Telefone: Código +244 pré-configurado
  • • Idioma: Português (Angola)
  • • Fuso Horário: WAT (Horário da África Ocidental)
